# Interactive Track Explorer

This page provides an interactive visualization of storm tracks generated by PyStormTracker. You can filter tracks by time, intensity, duration, and displacement, and switch between different map projections.

## About the Data
The tracks shown above are from the ERA5 Mean Sea Level Pressure (MSL) dataset for the DJF 2025-2026 period, tracked using the Hodges algorithm at 2.5° resolution.

### Filtering Options
*   **Map Projection**: Choose between Global, Globe (3D), Northern Hemisphere, or Southern Hemisphere.
*   **Time Range**: Select a specific time window within the dataset.
*   **Max Peak MSL**: Filter tracks by their minimum (peak) central pressure.
*   **Min Duration**: Only show tracks that lasted longer than the specified hours.
*   **Min Displacement**: Only show tracks that traveled further than the specified distance (great-circle distance between start and end points).
